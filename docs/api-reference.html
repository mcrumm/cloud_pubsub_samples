    cloud_pubsub_samples v1.0.0-dev API Reference

    Modules

    CloudPubsubSamples

    This Elixir app contains samples for working with Google Cloud Pub/Sub.

    CloudPubsubSamples.Pipeline

    A Broadway pipeline for consuming Cloud Pub/Sub messages.

    Mix Tasks

    mix publisher.create

    Creates a Google Cloud Pub/Sub topic on the current project.

    mix publisher.delete

    Deletes a Google Cloud Pub/Sub topic on the current project.

    mix publisher.list

    Lists the Google Cloud Pub/Sub topics on the current project.

    mix publisher.publish

    Publishes messages to a Google Cloud Pub/Sub topic.

    mix subscriber.create

    Creates a Google Cloud Pub/Sub subscription for the given topic.

    mix subscriber.delete

    Deletes a Google Cloud Pub/Sub subscription on the current project.

    mix subscriber.list_in_project

    Lists the Google Cloud Pub/Sub subscriptions for all topics in the current project.

    mix subscriber.list_in_topic

    Lists the Google Cloud Pub/Sub subscriptions for the current project.

    mix subscriber.start

    Starts listening for messages on the given subscription.
